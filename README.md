### âš ï¸ IMPORTANT: How to activate the Snake Animation

In the code above, I added a link to a "Snake" animation (the green snake eating your contribution squares). **This will be broken** until you set up a GitHub Action.

**Do this to fix it:**

1.  In your repository (the one named `judhayaprakash27052001`), click on the **Actions** tab.
2.  Click **New Workflow** -\> **set up a workflow yourself**.
3.  Name the file `snake.yml`.
4.  Paste this code inside that file:

<!-- end list -->

```yaml
name: Generate Datas

on:
  schedule: # execute every 12 hours
    - cron: "* */12 * * *"
  workflow_dispatch:

jobs:
  build:
    name: Jobs to update datas
    runs-on: ubuntu-latest
    steps:
      # Snake Animation
      - uses: Platane/snk@v3
        with:
          github_user_name: ${{ github.repository_owner }}
          outputs: |
            dist/github-contribution-grid-snake.svg
            dist/github-contribution-grid-snake-dark.svg?palette=github-dark
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}

      - uses: crazy-max/ghaction-github-pages@v3.1.0
        with:
          target_branch: output
          build_dir: dist
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}

```
